Common 3D Printing Problems

Click a problem below to see how to fix it.

1. Filament Clog

How to fix:

  • Heat the nozzle to printing temperature.
  • Try manually pushing filament through.
  • Perform a “cold pull.”
  • Clean or replace the nozzle if needed.
2. Bed Adhesion Issues

How to fix:

  • Clean the bed with alcohol.
  • Re-level the bed and adjust Z-offset.
  • Use glue stick or hairspray if needed.
  • Print the first layer slower.
3. Warping

How to fix:

  • Use a heated bed.
  • Increase bed temperature slightly.
  • Use an enclosure to keep heat in.
  • Add a brim or raft.
4. Layer Shifting

How to fix:

  • Tighten belts on X and Y axes.
  • Ensure printer is on a stable surface.
  • Reduce print speed slightly.
5. Stringing

How to fix:

  • Increase retraction distance.
  • Increase retraction speed.
  • Lower nozzle temperature slightly.
6. Under-Extrusion

How to fix:

  • Check for partial nozzle clogs.
  • Increase extrusion multiplier slightly.
  • Ensure filament spool moves freely.
